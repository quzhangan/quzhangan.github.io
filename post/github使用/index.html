Git和Git分别是什么#

  1. Git是一个运行在电脑上的版本控制软件： 一个代码可以提交多个版本，当你发现代码有问题需要回退，就可以恢复到之前的版本
  2. Git hub是基于Git这个版本控制软件打造的网站
  3. Git的三个概念：提交commit、仓库repository、分支branch
  4. git 下载链接：https://registry.npmmirror.com/binary.html?path=git-for-windows/

初始化一个仓库#

git clone 和点击下载的区别#

git clone下来的是一个仓库，而download zip只下载得到了文件夹。 文件夹要通过git初始化才能成为仓库

git 初始化#

在文件夹目录下打开git bash输入git init

git常用命令#

git init // 初始化本地仓库 git config –global user.name “myname” // 配置用户名 git config –global user.email “myname@mymail.com” //配置邮箱

git commit // 确认本地缓存区的内容，可以准备push git status // 确定本地缓存区的状态 git log –stat //查看提交的历史

第一次提交#

  1. git add -A 从工作区提交到暂存区
  2. git commit -m “提交信息 ” 从暂存区到仓库

项目维护#

git checkout //工作区打回去 git reset HEAD^ //提交后撤回

分支#

以当前分支为基础新建分支：git checkout -b 列举所有的分支：git branch 单纯地切换到某个分支：git checkout 删掉特定的分支：git branch -D 合并分支：git merge

git 与github远程仓库#

git remote add origin code@github.git //绑定本地和远程仓库 git pull // 拉取远程仓库的变化来同步本地的状态 git add // 确认本地仓库的变化到本地缓存区 git push // 提交本地仓库到远程仓库
