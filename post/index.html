Hugo搭建个人博客  [draft]

安装hugo 官网：https://gohugo.io/ 新建站点 hugo new site hugoblog 主题下载 主题官网：https://themes.gohugo.io/ 打开hugoblog文件夹下的themes目录 输入git clone 加克隆网址 会在该文件夹新建一个主题文件夹 打开config.toml文件添加 theme = '主题名' 就可以更改博客主题 文章编写 新建文章 hugo new post/名称.md 目录设置：在每个content目录下文件夹新建一个 _index.md 可以用来控制 标题和展示 可以在目录archetypes/default.md 下使用模板，即可在新建文章的时候自动加上模板配置 博客启动 hugo server -D //本地预览 将个人博客部署到远端服务器 打开GitHub 新建仓库 用户名.github.io 博客目录下输入 hugo --theme=hugo-PaperMod --baseUrl="https://用户名.github.io/" --buildDrafts 成功后新建了一个public文件夹 将内容传到GitHub 首先在public文件夹 git init git add . git commit -m "我的hugo博客第一次提交" 关联远端仓库 git remote add origin http://github.com/用户名/用户名.github.io.git 推到远端 git push git push -u origin master
